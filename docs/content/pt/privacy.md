# API MuscleLib – Política de Privacidade

## 1. Coleta de Dados
A API MuscleLib **não coleta**, **não armazena** e **não processa** dados pessoais dos usuários.

## 2. Cookies
Os cookies são utilizados exclusivamente para armazenar:
- Preferência de idioma
- Preferência de tema (claro / escuro / automático)

Nenhum cookie de rastreamento, análise ou publicidade é utilizado.

## 3. Compartilhamento de Dados
Nenhuma informação é compartilhada com terceiros.  
Os dados armazenados servem apenas para melhorar a experiência do usuário.

## 4. Alterações nesta Política
Esta Política de Privacidade pode ser atualizada periodicamente.  
A versão mais recente estará sempre disponível na documentação.
